# Geometry Library 
## Description
**Geometry** is a simple C# library for working with 2D geometry.

It allows you to:
- work with vectors (Vector)
- calculate vector and segment lengths
- add vectors
- check whether a point lies on a segment

## Installation
### Option 1: Clone the repository
```
git clone https://github.com/YekaterynaBoiko/Geometry.git
```
Navigate to the project folder:
```
cd Geomentry
```
Open the solution file:
```
Geometry.sln
```
in **Visual Studio** or **JetBrains Rider**

### Option 2: Add as a project reference
1. Open your C# project
2. Right-click on the project ->

> Add -> Project Reference

3. Select the *Geometry* project.

### Option 3: Using DLL (if you build the library)
After building the project? find the file:
```
bin/Debug/netX.X/Geometry.dll
```
and reference it in your project.

## Usage
### Import namespace

    using Geometry
